# Post-Mission Checklist

## After-Action Review and Equipment Maintenance

Use this checklist after tactical operations to document lessons learned and
ensure equipment is ready for the next mission.

### Team Debrief

* [ ] Debrief the team and document lessons learned
* [ ] Review mission objectives and outcomes
* [ ] Discuss communication effectiveness
* [ ] Identify areas for improvement
* [ ] Recognize successful tactics and procedures
* [ ] Address any safety concerns or incidents

### Documentation

* [ ] Submit mission logs and reports
* [ ] Compile event timeline with timestamps
* [ ] Document radio performance and range issues
* [ ] Note dead zones or communication difficulties
* [ ] Record equipment failures or malfunctions
* [ ] Photograph or diagram significant events if applicable

### Equipment Inspection

* [ ] Inspect and replenish all equipment
* [ ] Check radios for physical damage
* [ ] Inspect antennas and connections
* [ ] Verify all buttons and controls functional
* [ ] Clean radio exterior and accessories
* [ ] Check headsets and audio accessories

### Battery Management

* [ ] Recharge all batteries immediately after mission
* [ ] Inspect batteries for damage or swelling
* [ ] Record battery performance during mission
* [ ] Test battery capacity if degradation suspected
* [ ] Store batteries properly (40-60% charge for long-term)
* [ ] Replace batteries showing reduced capacity

### Radio Maintenance

* [ ] Clean radio and accessories thoroughly
* [ ] Inspect for moisture ingress (IP67 rating)
* [ ] Check programming and settings retained
* [ ] Test all programmed functions
* [ ] Verify emergency alarm operation
* [ ] Update firmware if updates available

### Configuration Review

* [ ] Verify radio programming unchanged
* [ ] Backup current configuration
* [ ] Review and update settings based on mission experience
* [ ] Document any configuration changes made
* [ ] Test modifications thoroughly before next mission

### Tactical Plan Updates

* [ ] Review and update tactical plans based on mission outcomes
* [ ] Incorporate lessons learned into procedures
* [ ] Update communication protocols if needed
* [ ] Revise equipment lists based on mission needs
* [ ] Update training requirements identified during mission

### Supply Replenishment

* [ ] Replenish spare batteries
* [ ] Replace damaged or worn accessories
* [ ] Restock first aid supplies
* [ ] Update maps and reference materials
* [ ] Prepare equipment for next deployment

### Administrative Tasks

* [ ] File incident reports if required
* [ ] Submit equipment repair requests
* [ ] Update training records
* [ ] Schedule follow-up training if needed
* [ ] Coordinate with support personnel

### Ready State Verification

* [ ] All equipment inspected and functional
* [ ] Batteries charged and ready
* [ ] Radios programmed correctly
* [ ] Documentation filed
* [ ] Team debriefed
* [ ] Ready for next mission

---

*This checklist should be completed within 24 hours of mission completion.*
