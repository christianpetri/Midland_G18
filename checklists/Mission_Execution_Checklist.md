# Mission Execution Checklist

## Active Operations Communication Protocol

Use this checklist during tactical operations to maintain effective communication
and situational awareness.

### Communication Discipline

* [ ] Maintain clear communication using designated call signs and protocols
* [ ] Listen before transmitting (use Monitor function - PF3)
* [ ] Keep transmissions brief and clear
* [ ] Use proper radio procedures (over, out, etc.)
* [ ] Respect emergency channel protocols
* [ ] Monitor for emergency traffic

### Status Updates

* [ ] Regularly update the team on progress and changes in situation
* [ ] Maintain scheduled check-in intervals (default: every 15 minutes)
* [ ] Report significant events immediately
* [ ] Confirm receipt of important messages
* [ ] Use standardized status codes if established

### Team Coordination

* [ ] Monitor and log team member locations and status
* [ ] Track team movements and positions
* [ ] Coordinate actions through designated leader
* [ ] Maintain awareness of all team member locations
* [ ] Update team on environmental changes

### Event Documentation

* [ ] Record key events and timestamps for after-action reviews
* [ ] Log communication difficulties or dead zones
* [ ] Note equipment issues or failures
* [ ] Document emergency situations and responses
* [ ] Record weather changes affecting operations

### Battery and Equipment Monitoring

* [ ] Monitor battery levels throughout mission
* [ ] Switch to spare batteries before depletion
* [ ] Report equipment malfunctions immediately
* [ ] Maintain radio in proper operating condition
* [ ] Keep antenna and connections secure

### Emergency Procedures

* [ ] Know emergency alarm activation (3-second hold)
* [ ] Maintain awareness of emergency frequencies
* [ ] Respond immediately to emergency traffic
* [ ] Follow established emergency protocols
* [ ] Coordinate emergency response with team

### Channel Management

* [ ] Stay on assigned primary channel unless directed
* [ ] Use backup channels only when authorized
* [ ] Avoid channel hopping without coordination
* [ ] Use scan function (PF4 long press) when monitoring multiple channels
* [ ] Verify channel before transmitting critical information

### Security Measures

* [ ] Use CTCSS/DCS codes to isolate team communications
* [ ] Enable scrambler for sensitive operations
* [ ] Maintain operational security in transmissions
* [ ] Avoid transmitting sensitive information in clear
* [ ] Follow established code words and phrases

---

*This checklist should be reviewed at regular intervals during the mission.*
