# Pre-Mission Checklist

## Tactical Team Pre-Mission Verification

Use this checklist before any tactical operation to ensure all team members are
properly equipped and briefed.

### Communication Equipment

* [ ] Verify all communication devices are operational
* [ ] Perform radio checks on all assigned channels
* [ ] Test emergency alarm functionality
* [ ] Verify CTCSS/DCS codes are coordinated across team
* [ ] Check battery levels (minimum 80% charge recommended)
* [ ] Prepare spare batteries (1600mAh packs)

### Team Coordination

* [ ] Conduct team briefing to discuss objectives and roles
* [ ] Assign call signs to each team member
* [ ] Establish emergency code phrases
* [ ] Set check-in intervals (default: every 15 minutes)
* [ ] Review communication protocols and procedures
* [ ] Designate primary and backup channels

### Equipment Verification

* [ ] Ensure all team members have required equipment:
  * [ ] Radios (Midland G18 PRO)
  * [ ] Batteries (spare packs)
  * [ ] First aid kits
  * [ ] Maps/GPS devices
  * [ ] Audio accessories (headsets/mics if applicable)

### Environmental Assessment

* [ ] Confirm weather conditions
* [ ] Review terrain and potential RF obstacles
* [ ] Identify elevated positions for better range
* [ ] Establish contingency plans for adverse conditions
* [ ] Note potential interference sources

### Radio Configuration Verification

* [ ] Frequency: PMR446 (446.00625 - 446.19375 MHz)
* [ ] Default Channel: 1 (446.00625 MHz) or team-assigned
* [ ] Power Mode: Low or High (both = 500mW ERP)
* [ ] CTCSS/DCS: Team-specific privacy codes enabled
* [ ] BCL: Enabled (if programmed)
* [ ] Scrambler: Enabled if privacy required
* [ ] TOT: Configured (recommended: 180 seconds)
* [ ] Emergency Button: Enabled and tested

### Final Verification

* [ ] All checklist items completed
* [ ] Team acknowledges readiness
* [ ] Backup communication plan established
* [ ] Mission timeline confirmed

---

*This checklist should be completed 30-60 minutes before mission start.*
