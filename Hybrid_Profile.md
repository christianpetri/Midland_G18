# Hybrid Profile for Midland G18 Radio

This profile balances tactical and hiking features for versatile use, ensuring reliable communication and ease of operation in various scenarios.

---

## Configuration

### General Settings:
1. **TX Power**:
   - Default to **Low Power** to conserve battery.
   - Include a toggle to **High Power** (via **PF4 Hold**) for long-range communication when needed.

2. **W/N (Wide/Narrow)**:
   - Set to **Narrow** for clearer communication and compliance with regulations.

3. **BCL (Busy Channel Lockout)**:
   - Enable **BCL** to avoid disrupting existing communications in crowded or shared channels.

### Button Assignments:
1. **PF3**: Monitor
   - Use this for manual monitoring of weak or distant signals when required.
   - Ideal for checking channel activity before transmitting.

2. **PF3 Hold**: FM Radio Toggle
   - Use this to enable or disable the FM radio for entertainment or weather updates during downtime.
   - FM radio remains off by default to conserve battery.

3. **PF4**: Scan
   - Activate scanning to monitor multiple channels for activity.
   - Useful for tactical situational awareness or finding the least busy channel during hiking.

4. **PF4 Hold**: Power Toggle
   - Switch between **Low Power** and **High Power** modes based on range requirements.

### Emergency Button:
- Enable the emergency alarm feature for quick activation in case of emergencies.
- Configure this to a hold function to prevent accidental activation.

### VOX:
- **Disabled** by default, as it is not suitable for loud environments or hearing protection.

### Scrambler:
- Enable if privacy or encrypted communication is required (optional based on use case).

---

## Why This Profile Works
- **For Tactical Use**:
  - High Power mode and manual monitoring ensure reliable communication in critical scenarios.
  - Channel scanning enhances situational awareness.
  - Emergency alarm supports quick action in emergencies.

- **For Hiking Use**:
  - Low Power mode conserves battery during long hikes.
  - FM radio provides entertainment or access to weather updates.
  - Monitoring weak signals helps in remote areas.
