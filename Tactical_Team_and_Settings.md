# Tactical Team Operations and Settings

## Tactical Team Checklist

### Pre-Mission Checklist
- [ ] Verify all communication devices are operational.
- [ ] Conduct a team briefing to discuss objectives and roles.
- [ ] Ensure all team members have the required equipment:
  - Radios
  - Batteries
  - First aid kits
  - Maps/GPS devices
- [ ] Perform radio checks on all assigned channels.
- [ ] Confirm weather conditions and contingency plans.

### Mission Execution Checklist
- [ ] Maintain clear communication using designated call signs and protocols.
- [ ] Regularly update the team on progress and changes in the situation.
- [ ] Monitor and log team member locations and status.
- [ ] Record key events and timestamps for after-action reviews.

### Post-Mission Checklist
- [ ] Debrief the team and document lessons learned.
- [ ] Inspect and replenish all equipment.
- [ ] Submit mission logs and reports.
- [ ] Review and update tactical plans based on mission outcomes.

## Settings for Tactical Operations

### Radio Configuration
- Frequency Range: PMR446 (446.0 - 446.2 MHz)
- Default Channel: 1 (446.00625 MHz)
- Channel Spacing: 12.5 kHz
- CTCSS/DCS Settings: Enabled (as per mission requirements)

### Communication Protocols
- Standard Call Signs: Assigned per team member
- Emergency Code: "RED ALERT"
- Check-In Interval: Every 15 minutes
- Silent Mode: Activated during stealth operations

### Equipment Maintenance
- Radio Batteries: Fully charged before deployment
- Signal Boosters: Pre-tested for range and clarity
- Headsets: Noise-cancelling models only
- Backup Units: 2 per team

### Security Measures
- Encrypted Channels: Mandatory for sensitive operations
- Jamming Devices: Deployed when necessary
- Access Codes: Rotated weekly

## Programmable Function (PF) Button Settings

### Default PF Button Assignments
1. **PF1 (Top Button)**: 
   - Function: Emergency Alarm
   - Description: Press and hold to send an emergency alert to all team members. This function must be enabled via programming software.

2. **PF2 (Middle Button)**:
   - Function: Monitor/Scan
   - Description: Press to monitor the current channel or start scanning for active channels.

3. **PF3 (Bottom Button)**:
   - Function: High/Low Power Toggle
   - Description: Switch between high and low power modes to save battery or boost signal.

### Customizable PF Button Options
- **Monitor**: Listen to the current channel without squelch.
- **Scan**: Cycle through all programmed channels.
- **Emergency Alarm**: Send a distress signal.
- **High/Low Power**: Adjust transmission power.
- **Volume Preset**: Set a fixed volume level for noisy environments.
- **Channel Lock**: Prevent accidental channel changes during operations.
