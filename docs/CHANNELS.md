# Midland G18 PRO - Complete Channel Reference

## Channel Overview

### Understanding "99 Channels"

The G18 PRO has **16 actual PMR446 frequencies**. The "99 channels" refers to
99 different channel configurations using those same 16 frequencies with various
CTCSS tone combinations.

- **16 Physical Frequencies**: 446.00625 - 446.19375 MHz (12.5 kHz spacing)
- **99 Channel Configurations**: 16 base + 83 pre-programmed with CTCSS tones

**How CTCSS Works:**

CTCSS (Continuous Tone-Coded Squelch System) adds an inaudible sub-audible tone
to transmissions. Your radio only opens the speaker when it receives the matching
tone. This creates "privacy" by filtering out other users on the same frequency.

**Example:**

- Channel 1 (446.00625 MHz, no CTCSS) and Channel 17 (446.00625 MHz, CTCSS
  114.8) use the **same physical frequency**
- A transmission on Channel 1 will be heard by Channel 1 users, but filtered out
  by Channel 17 users (wrong tone)
- Both are sharing the same airwave - CTCSS just filters who you hear

**Channel Breakdown:**

- **Channels 1-16**: Base PMR446 channels with selectable CTCSS/DCS
- **Channels 17-99**: Same 16 frequencies with pre-programmed CTCSS tones for
  convenience

## Notes

### Understanding CTCSS vs Physical Channels

**Key Point**: CTCSS (Continuous Tone-Coded Squelch System) does **not** create
separate channels - it only filters which transmissions you hear on the same
frequency.

**Technical Reality:**

- All "channels" using the same frequency (e.g., 446.00625 MHz) share the same
  radio spectrum
- CTCSS tones are sub-audible (below 300 Hz) and don't interfere with voice
- Your radio transmits on the physical frequency AND adds the CTCSS tone
- Other radios on that frequency hear the transmission, but only those with
  matching CTCSS settings will open their speaker

**Privacy Limitations:**

- CTCSS provides "selective calling" not encryption or true privacy
- Anyone on the same frequency without CTCSS filtering will hear all
  transmissions
- Scanner users and other PMR446 radios can monitor all traffic on the base
  frequency
- For actual security, use the scrambler feature (though it's not
  military-grade encryption)

**Practical Use:**

- **Channels 1-16**: Use when you want maximum compatibility or need to hear all
  traffic on a frequency
- **Channels 17-99**: Use when multiple groups share the same area and you want
  to filter out other groups' conversations
