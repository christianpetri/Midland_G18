# General Frequency Table for Midland G18 Radio

This table includes commonly used frequencies for general, tactical, and hiking use. Adjust these frequencies according to local regulations and needs.

| Channel | Frequency (MHz) | Usage Notes                |
|---------|------------------|----------------------------|
| 1       | 462.5625         | General communication     |
| 2       | 462.5875         | Tactical short-range use  |
| 3       | 462.6125         | Long-range communication  |
| 4       | 462.6375         | Emergency coordination    |
| 5       | 462.6625         | Hiking group communication|
| 6       | 462.6875         | Weather updates           |
| 7       | 462.7125         | Reserved for FM radio     |

---

### Notes
1. **Channel 4 (Emergency Coordination)**:
   - Reserved for emergencies and should be used sparingly.
   - Ensure the emergency alarm feature is active when using this channel.
   
2. **Channel 6 (Weather Updates)**:
   - Tune to this channel for NOAA weather updates or other regional weather services.
   
3. **Channel 7 (FM Radio)**:
   - Non-critical communication or entertainment.

---

## Instructions
1. Program these frequencies into your Midland G18 using the programming software.
2. Test each channel to ensure proper functionality and compliance with local regulations.
