# Midland G18 Radio Documentation

This repository contains user manuals and programming guides for the Midland G18 radio. It includes resources in both English and German to assist with usage, programming, and troubleshooting.

## Manuals
- **[English Manual](docs/Midland_G18_Manual_EN.pdf)**: Detailed guide for operating the Midland G18 radio.
- **[German Manual](docs/Midland_G18_Manual_DE.pdf)**: Anleitung für die Bedienung des Midland G18 Funkgeräts.

## Features of the Midland G18
- PMR446 Transceiver
- Long-range communication
- Programming options for CTCSS, TOT, VOX, Roger Beep, Squelch, and more.

## How to Use
1. Download the relevant manual from the `docs/` folder.
2. Follow the instructions within the manual for setting up and programming the radio.
3. Refer to additional programming software guides if required (linked in the manuals).
